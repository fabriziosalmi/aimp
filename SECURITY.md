# Security Policy

## Supported Versions

Only the latest release of AIMP is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of AIMP seriously. If you believe you have found a security vulnerability, please do NOT open a public issue.

Instead, please report it to the maintainers privately via [GitHub Security Advisories](https://github.com/fabriziosalmi/aimp/security/advisories/new).

Please include:
- A description of the vulnerability.
- Steps to reproduce (POC).
- Potential impact.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue before a public disclosure is made.
